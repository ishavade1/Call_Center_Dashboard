#### **1. Overview**



This project analyzes call center performance to identify top-performing agents, call trends, and customer satisfaction patterns. The dashboard helps management make data-driven decisions to improve sales and service efficiency.



#### **2. Dataset**



Columns include:



* Agent ID / Representative
* Total Calls
* Call Duration
* Average Rating
* Happy Calls
* Location (Cincinnati, Cleveland, Columbus)
* Format: Excel (.xlsx)



#### **3. Tools Used**



* Microsoft Excel
* Pivot Tables
* Slicers \& Filters
* Charts \& Graphs
* Conditional Formatting



#### **4. Analysis Steps**



* Cleaned and structured raw call data.
* Calculated KPIs: total calls, total amount, duration, avg. rating, happy calls.
* Built monthly call trend line chart.
* Analyzed Gender vs Location trends.
* Created interactive filters by representative \& location.
* Highlighted top-performing agents for recognition.



#### **5. Dashboard Features**



* **KPIs:** Calls, Total Amount, Duration, Avg. Rating, Happy Calls
* **Charts:** Call trends, Gender vs Location, Ratings distribution
* **Filters:** Dynamic selection of agents \& locations
* **Tables:** Performance by location \& representative
* **Highlights:** Top 3 agents, ranking by calls and revenue



#### **6. Key Insights**



* Identified top-performing agents for incentives.
* Peak months for calls identified to optimize staffing.
* Gender and location analysis helps in customer service planning.



#### **7. Business Impact**



* Provides actionable insights to improve call center operations.
* Supports strategic decisions for agent performance, training, and incentives.
* Enhances reporting efficiency and transparency.
